Caso real: Site de jogos:

Para criar um site de jogos de qualidade, é importante utilizar o Flexbox para melhorá-lo. Esse site apresenta diferentes jogos organizados em cards, que contém a capa do jogo, o nome e o gênero dele. Além disso, eles devem estar organizados lado a lado em telas como de computador e tablet, e em coluna caso o site seja acessado por uma tela menor, como o celular. Utilizando o flexbox, o contêiner que envolve os itens do site deve ser colocado como um elemento flexível, para organizar os cards dos jogos.
