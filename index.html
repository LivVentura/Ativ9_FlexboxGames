Reflexão e Aplicação Prática

Propriedades CSS utilizadas durante os jogos:

  • Justify-content: alinha os itens, por padrão, horizontalmente. Os valores possíveis para o justify-content são: center(alinha horizontalmente no centro); flex-start(itens alinhados do lado esquerdo do contêiner); flex-end(itens alinhados do lado direito do contêiner); space-between(itens com espaço igual entre eles) e space-around(itens organizados com um espaço igual ao redor deles).
  • Align-items: essa propriedade, por padrão, alinha os itens verticalmente. Os valores possíveis para o Align-items são: center(alinha os itens no centro verticalmente); flex-start(os itens são alinhados no topo do contêiner); flex-end(alinha os itens na parte inferior do contêiner); stretch(desde que não tenham altura definida os itens são esticados para caber no contêiner) e baseline(alinhamento pela linha de base do contêiner).
  • Flex-direction: a declaração flex-direction refere-se a direção, ou seja, se um contêiner renderiza seus itens horizontalmente ou verticalmente. Os valores possíveis para o Flex-direction são: row(é o padrão, em que os itens são dispostos na mesma direção que o texto); row-reverse(os itens são dispostos na direção oposta da direção do texto); column(os itens são dispostos em coluna, de cima para baixo) e column-reverse(os itens são dispostos em coluna, de baixo para cima).

Caso real: Site de jogos:

Para criar um site de jogos de qualidade, é importante utilizar o Flexbox para melhorá-lo. Esse site apresenta diferentes jogos organizados em cards, que contém a capa do jogo, o nome e o gênero dele. Além disso, eles devem estar organizados lado a lado em telas como de computador e tablet, e em coluna caso o site seja acessado por uma tela menor, como o celular. O site seria um contêiner principal e um contêiner que envolve a seção onde os jogos estão. Utilizando o flexbox, o contêiner que envolve os itens do site deve ser colocado como um elemento flexível, para organizar os cards dos jogos em linhas e o menu do site. Para colocar espaçamento entre os cards dos jogos e ao redor deles, a propriedade Justify-content: space-around pode ser utilizada. Utilizaria também a propriedade flex-wrap para organizar os cards dos jogos corretamente na tela e quebrar linha caso necessário. Portanto, utilizaria o flexbox para organizar o layout da página, suas propriedades seriam necessárias para melhorar as disposições dos elementos na páginas e controlar suas ações da melhor forma possível
